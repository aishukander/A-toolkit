如果想要自己打包程式請先以```pip install pyinstaller```安裝pyinstaller並且下載該專案後到所需小工具所在的資料夾開啟終端機並且輸入```pyinstaller -F (小工具名稱.py)```以此來將小工具打包並且輸出到dist資料夾 [ 記得把指令的()去掉 ]
